# Changelog

All notable changes to the Higgsfield Cursor plugin are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0]

### Added

- Initial release.
- Remote HTTP MCP server wired to `https://mcp.higgsfield.ai/mcp`.
- Skills: `generate-image`, `generate-video`, `virality-prediction`, `media-management`.
- Rule: `higgsfield-usage` — agent guardrails for using the Higgsfield MCP reliably.
- Slash command: `/higgs` — natural-language entrypoint.
